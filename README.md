# LegiAgent: Law-Governed Multi-Agent Evolution via Deterministic Manifold Projection

This repository contains the official implementation of **LegiAgent**, a framework designed to ensure legal compliance in Large Language Model (LLM) based multi-agent simulations through Deterministic Manifold Projection (DMP).

---

## üöÄ Overview
LegiAgent bridges the gap between natural language tax statutes and executable agent behaviors. By mapping legal clauses into a geometric manifold, we provide a mathematical guarantee for agent compliance in complex socio-economic environments.

- `src/simulation/`: Multi-agent simulation environment (fiscal and tax scenarios).
- `data/`: Sample regulatory datasets and anonymized agent configuration files.
- `scripts/`: Shell scripts to reproduce the experiments in Section 5.
